## üìå Abstract

Traditional AI-based cancer detection models often rely on a **single data modality**, which limits their accuracy. This review investigates **multi-modal integration** ‚Äî combining different types of biomedical data ‚Äî and shows how it significantly improves diagnostic performance across various cancer types.

---

## üîç Key Contributions

- Overview of **10 recent peer-reviewed studies** (2025) from ScienceDirect
- Comparison of single-modal vs multi-modal models
- Analysis of **bias**, data fusion strategies, and model performance
- Survey of models like **GeneXAI, CAFNet, OmniFuse**, and others
